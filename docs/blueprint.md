# **App Name**: BIOVALOUR

## Core Features:

- Real-time Sensor Data Display: Visualize live temperature, pH, flowrate, BOD, COD, turbidity, conductivity, gas levels (ppm), and humidity readings in a dashboard format.
- Automated Reactor Status Indicator: Dynamically display the reactor's operational status (SAFE, WARNING, ALERT) based on predefined sensor data thresholds.
- Threshold-based Alert System: Trigger immediate visual alerts and notifications when critical temperature, pH, or gas level thresholds are exceeded.
- Historical Data Logging & Retrieval: Securely store and retrieve past sensor data, allowing users to review trends and historical performance.
- Daily PVC Throughput Reporting: Display the calculated or reported daily quantity of PVC waste processed.
- AI-powered Anomaly Detection Tool: Utilize AI to analyze sensor data for unusual patterns and predict potential system issues or threshold breaches before they occur.
- Stirred Tank Bioreactor Visualization: Provide an interactive, real-time visual representation of the stirred tank bioreactor, including embedded sensor locations and their current readings.

## Style Guidelines:

- Primary brand color: A deep, authoritative blue (#1741B2) reflecting technological reliability and environmental focus.
- Background color: A subtle, desaturated light blue-gray (#EDF2F2) to maintain a clean, professional dashboard appearance and highlight alerts.
- Accent color: A vibrant aqua (#34CBDE) used for interactive elements and drawing attention to key information.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly legible appearance suitable for data-heavy displays.
- Utilize clean, modern line icons for data points and dashboard controls. Implement clear, color-coded status icons (e.g., green for safe, orange for warning, red for alert) consistent with industry standards. Include specific icons for bioreactor components and sensor types (pH, flowrate, BOD, etc.).
- Employ a responsive grid-based layout for data cards, ensuring optimal display across various devices. Prioritize clarity and easy access to critical status information, with a dedicated section for the bioreactor visualization.
- Implement subtle, smooth transitions for real-time data updates and state changes, providing a seamless user experience without distraction. Critical alerts could have a more prominent, but brief, animation. Include subtle animations for the bioreactor visualization to show flow or stirring.
